### Usage

Given two values, returns true if they are equal.

```ts
import * as _ from 'radashi'

_.isEqual(null, null) // => true
_.isEqual([], []) // => true

_.isEqual('hello', 'world') // => false
_.isEqual(22, 'abc') // => false
```
