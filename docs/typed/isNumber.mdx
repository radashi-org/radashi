### Usage

Pass in a value and get a boolean telling you if the value is a number.

```ts
import * as _ from 'radashi'

_.isNumber('hello') // => false
_.isNumber(['hello']) // => false
_.isNumber(12) // => true
_.isNumber(NaN) // => true
```
