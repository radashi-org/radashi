### Usage

Checks if the given value is primitive.

Primitive Types: number , string , boolean , symbol, bigint, undefined, null

```ts twoslash
import * as _ from 'radashi'

_.isPrimitive(22) // => true
_.isPrimitive('hello') // => true
_.isPrimitive(['hello']) // => false
```
