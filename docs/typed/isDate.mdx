### Usage

Determine if a value is a Date. Does not check that the input date is valid, only that it is a Javascript Date type.

```ts
import * as _ from 'radashi'

_.isDate(new Date()) // => true
_.isDate(12) // => false
_.isDate('hello') // => false
```
