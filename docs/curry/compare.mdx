## Basic usage

Create a function that compares two values. It accepts either a getter or a property name. Optionally, you can pass a custom compare function.

```ts
import * as _ from 'radash'

const users = [
  { id: 1, firstName: 'Alice', lastName: 'Smith' },
  { id: 3, firstName: 'Charlie', lastName: 'Brown' },
  { id: 2, firstName: 'Drew', lastName: 'Johnson' },
]

const compareById = _.compare('id')
users.sort(compareById)
// [Alice, Drew, Charlie]

const compareByFullName = _.compare(
  (user) => `${user.firstName} ${user.lastName}`,
  (a, b) => b.localeCompare(a),
)
users.sort(compareByFullName)
// [Alice, Charlie, Drew]
```
