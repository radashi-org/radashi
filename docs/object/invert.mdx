---
title: invert
description: Invert the keys and values of an object
---

## Basic usage

Given an object returns a new object with the keys and values reversed.

```ts
import * as _ from 'radashi'

const powersByGod = {
  ra: 'sun',
  loki: 'tricks',
  zeus: 'lighning',
}

_.invert(powersByGod) // => { sun: ra, tricks: loki, lightning: zeus }
```
