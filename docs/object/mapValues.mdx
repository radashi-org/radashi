---
title: mapValues
description: Map over the keys of an object
since: 12.1.0
---

### Usage

Given an object and a `toValue` callback function, returns a new object with all the values
mapped through the `toValue` function. The callback is given both the value and key for each entry.

```ts twoslash
import * as _ from 'radashi'

const ra = {
  mode: 'god',
  power: 'sun',
}

_.mapValues(ra, value => value.toUpperCase()) // => { mode: 'GOD', power: 'SUN' }
_.mapValues(ra, (value, key) => key) // => { mode: 'mode', power: 'power' }
```
