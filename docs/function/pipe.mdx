title: pipe
description: 'Applies a series of functions to an initial value in sequence'

### Usage

Pass in an initial value followed by a sequence of functions. The `pipe` function will apply each function in sequence, passing the result of the previous function to the next. It returns the final result without modifying the original value.

```ts
import * as _ from 'radashi'

const result = _.pipe(
  5,
  x => x * 2, // 10
  x => x.toString(), // "10"
  x => x + '!', // "10!"
)
console.log(result) // => "10!"

const transformedString = _.pipe(
  'hello',
  x => x.toUpperCase(), // "HELLO"
  x => x.split('').reverse().join(''), // "OLLEH"
)
console.log(transformedString) // => "OLLEH"

const filteredArray = _.pipe(
  [1, 2, 3],
  arr => arr.map(x => x * 2), // [2, 4, 6]
  arr => arr.filter(x => x >= 4), // [4, 6]
  arr => arr.join('-'), // "4-6"
)
console.log(filteredArray) // => "4-6"
```

`pipe` can also be used for complex data transformations, such as working with objects and filtering large datasets.
