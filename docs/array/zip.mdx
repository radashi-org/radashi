title: zip

---

### Usage

Creates an array of grouped elements, the first of which contains the first elements of the given arrays, the second of which contains the second elements of the given arrays, and so on.

```ts
import * as _ from 'radashi'

const names = ['ra', 'zeus', 'loki']
const cultures = ['egypt', 'greek', 'norse']

_.zip(names, cultures)
// => [
//   [ra, egypt]
//   [zeus, greek]
//   [loki, norse]
// ]
```

### Search terms

- Often called `combine`, or `group` in other libraries.

### Popular use cases

- **Data Transformation**: Combine multiple arrays of related data points into a single array of tuples, facilitating easier access and manipulation, such as transforming user information arrays into structured data for further processing.
- **Multi-Array Pairing**: Create grouped arrays where elements from corresponding indices are collected together, useful in rendering multi-dimensional datasets in UI components like tables or lists.
- **Coordinate Pairing**: Zip together separate arrays of x and y coordinates for graphical representations, such as plotting points on a chart or in a graphical application.
