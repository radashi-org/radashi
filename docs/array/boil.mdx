### Usage

Given an array of items return the final item that wins the comparison condition. Useful for more complicated min/max.

```ts twoslash
import * as _ from 'radashi'

const gods = [
  {
    name: 'Ra',
    power: 100,
  },
  {
    name: 'Zeus',
    power: 98,
  },
  {
    name: 'Loki',
    power: 72,
  },
]

const boilByPower = _.boil(gods, (a, b) => (a.power > b.power ? a : b))
// => { name: 'Ra', power: 100 }
```
