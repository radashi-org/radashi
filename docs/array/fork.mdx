### Usage

Given an array of items and a condition, returns two arrays where the first contains all items that passed the condition and the second contains all items that failed the condition.

```ts
import * as _ from 'radashi'

const gods = [
  {
    name: 'Ra',
    power: 100,
  },
  {
    name: 'Zeus',
    power: 98,
  },
  {
    name: 'Loki',
    power: 72,
  },
  {
    name: 'Vishnu',
    power: 100,
  },
]

const [finalGods, lesserGods] = _.fork(gods, f => f.power > 90) // [[ra, vishnu, zeus], [loki]]
```

### Search terms

- Commonly referred to as `partition`, `split`, `divide`, or `groupBy` in other libraries.

### Popular use cases

- Splitting an array of numbers into even and odd values for mathematical operations.
- Dividing a list of users into verified and unverified categories based on a boolean flag.
- Grouping an array of objects by a certain attribute (e.g., active vs inactive status).
- Filtering data for analytics by separating required and optional fields.
- Organizing items in a store based on stock availability (in-stock vs out-of-stock).
