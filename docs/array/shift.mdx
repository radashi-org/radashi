---
title: shift
description: Shift array items by n steps
---

### Usage

Given a list of items, return an array that shift right n positions.

```ts
import * as _ from 'radashi'
const arr = [1, 2, 3, 4, 5, 6, 7, 8, 9]
_.shift(arr, 3) // => [7, 8, 9, 1, 2, 3, 4, 5, 6]
```

### Search terms

- Often called `rotate`, `cycle`, or `reorder` in other libraries.

### Popular use cases

- **Data Circular Shifting**: Use `shift` to manipulate data structures by rearranging elements in circular lists or queues, ensuring optimal accessibility to the most recent data.
- **User Interface Navigation**: Implement in UI components (e.g., carousels or sliders) to smoothly transition between items, enhancing user interaction experiences.
- **Game Development**: Apply when managing player turns or item inventories, effectively rotating elements display based on game mechanics or player actions.
