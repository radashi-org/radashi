title: counting
description: Creates an object with counts of occurrences of items
since: 12.1.0
---

### Usage

Given an array of objects and an identity callback function to determine
how each object should be identified. Returns an object where the keys
are the id values the callback returned and each value is an integer
telling how many times that id occurred.

```ts
import * as _ from 'radashi'

const gods = [
  {
    name: 'Ra',
    culture: 'egypt',
  },
  {
    name: 'Zeus',
    culture: 'greek',
  },
  {
    name: 'Loki',
    culture: 'greek',
  },
]

_.counting(gods, g => g.culture) // => { egypt: 1, greek: 2 }
```

### Search terms

- Often called `countBy`, `groupBy`, or `frequencyCounter` in other libraries.

### Popular use cases

- Counting the frequency of unique items in an array, such as tallying votes or survey responses.
- Grouping data by categories, e.g., classifying numbers as odd/even or categorizing objects by type (e.g., counting fruits by variety).
- Analyzing data distributions, such as counting occurrences of words in text or responses in a dataset.
- Summarizing metrics in analytics, like counting the number of clicks by user type or categorizing sales by region.
- Generating statistical data for reports, including item frequency in inventory management or web traffic analysis by source.
