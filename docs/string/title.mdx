### Usage

Formats the given string in title case fashion

```ts twoslash
import * as _ from 'radashi'

_.title('hello world') // => 'Hello World'
_.title('va_va_boom') // => 'Va Va Boom'
_.title('root-hook') // => 'Root Hook'
_.title('queryItems') // => 'Query Items'
```
