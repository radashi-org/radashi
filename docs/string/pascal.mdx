### Usage

Formats the given string in pascal case fashion.

```ts twoslash
import * as _ from 'radashi'

const hw = _.pascal('hello world') // => 'HelloWorld'
const vvb = _.pascal('va va boom') // => 'VaVaBoom'
const h2 = _.pascal('helloWorld') // => 'HelloWorld'
```
